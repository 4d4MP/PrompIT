# PrompIT

PrompIT is a lightweight macOS teleprompter that stays on top of other windows and shows your script in a clean, distraction-free view. The app uses a floating, borderless window to keep your text visible while you present or record.

## Features

- Floating window that stays above other apps.
- Full-screen friendly: appears across all Spaces and alongside full-screen apps.
- Simple, minimal UI focused on readability.

## Requirements

- macOS 13 or later.
- Xcode 15 or later.

## Getting Started

1. Open the project in Xcode.
2. Build and run the `PrompIT` target.
3. Add your script in the app settings to start prompting.
